一些（非正式的）编码英语单词规范

Posted on 2018-02-27 Edited on 2018-11-19

来源于Jenkins、Sonar等，一些名称的命名建议

业务相关

  • culprit Jenkins构建成功后，会发送邮件到最后一次构建成功的提交者
  • set 集合
  • actions 操作

ID、名称等

  • userId
  • userName
  • displayName
  • fullDisplayName
  • shortDescription

密码

  • obfuscatedPassword 模糊的密码（用星号替代）

文件

  • fileName
  • displayPath
  • relativePath

时间等

  • duration 持续时间
  • estimatedDuration 预估时间

  • startedAt 开始时间
  • executedAt 执行时间
  • timeMs 毫秒时间
  • uptime 工作总时长
  • interval 时间间隔

日志

  • keepLog 是否记录日志

  • artifact 产品

动作

  • analysis 分析
  • obtain 获取
  • suggestion 建议

状态

  • status SUCCESS

  • arch 架构
